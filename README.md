**Kaza** is a microservices-ready housing and rental platform built with **Java Spring Boot** and **PostgreSQL**. It allows landlords to upload properties for rent, and tenants to search, view, and inquire about available listings. This project demonstrates production-level concepts including REST APIs, database integration, Dockerization, and modular architecture.  

---

## Features

- **User Management**: Tenants and landlords can register, log in, and manage profiles.  
- **Property Management**: Landlords can add, update, and manage property listings with details and images.  
- **Property Search**: Tenants can search and filter properties by location, price, and availability.  
- **Inquiries**: Tenants can send inquiries or booking requests to landlords.  
- **Role-Based Access**: Basic support for different user roles (Tenant, Landlord).  
- **Dockerized**: Easy setup for local development using Docker and Docker Compose.  

---

## Tech Stack

- **Backend**: Java 21, Spring Boot  
- **Database**: PostgreSQL  
- **ORM**: Spring Data JPA  
- **Security**: Spring Security (JWT ready for future implementation)  
- **Validation & Utilities**: Lombok, Bean Validation  
- **Dev Tools**: Spring Boot DevTools for hot reload  
- **Containerization**: Docker
